Where to go from here

Our AI can play the game but it is my no means perfect. With more time the areas we would build on would include:

  • Convert initial placement to a series of moves based on default placement.
  • Utilise ‘held’ piece function (store a piece for later use, retrieve piece for next round).
  • Optimise the game functions to run episodes.
  • Changing size of tetris grid to a larger size. (train random sizes and play random sizes that are different to trainied).
  • Add CNN so it can see the game state rather than being fed the game state.
